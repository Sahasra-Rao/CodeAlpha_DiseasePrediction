# Disease Prediction from Medical Data using Machine Learning

## Objective
To predict whether a patient is likely to have diabetes based on medical data using machine learning techniques.

## Dataset
Pima Indians Diabetes Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Algorithm
- Random Forest Classifier

## Features Used
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## Project Workflow
1. Data Loading and Exploration
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Model Evaluation
6. Prediction on New Patient Data

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Classification Report
- Confusion Matrix

## Conclusion
The model successfully predicts the likelihood of diabetes using patient medical data and demonstrates the application of machine learning in healthcare.
